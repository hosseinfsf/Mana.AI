# Mana.AI - دستیار هوشمند همه‌کاره ایرانی

Mana.AI یک اپلیکیشن فلاتر است که با هدف ارائه یک دستیار هوشمند همه‌کاره ایرانی توسعه یافته است. این پروژه شامل ویژگی‌های متنوعی از جمله مدیریت وظایف، ارائه محتوای معنوی (مانند اشعار حافظ)، ریتم صبحگاهی و ادغام با مدل‌های هوش مصنوعی است.

## ویژگی‌های اصلی

- صفحه آغازین (Splash Screen)
- صفحه معرفی (Onboarding)
- صفحه خانه (Home Screen)
- مدیریت وظایف (Tasks Screen)
- ویژگی حافظ (Hafez Screen)
- صفحه مراسم صبح (Morning Mana Screen)
- صفحه تنظیمات (Settings Screen)
- ادغام با Gemini API برای قابلیت‌های هوش مصنوعی
- پشتیبانی از تم روز/شب
- ذخیره سازی محلی داده‌ها
- قابلیت‌های فارسی (RTL)

## وابستگی‌های کلیدی

- Flutter (SDK >=3.0.0 <4.0.0)
- google_generative_ai: برای ارتباط با مدل‌های گوگل
- provider: برای مدیریت وضعیت
- shared_preferences: برای ذخیره سازی محلی
- flutter_svg, lottie: برای گرافیک‌های پویا
- persian_datetime_picker: برای تقویم فارسی
